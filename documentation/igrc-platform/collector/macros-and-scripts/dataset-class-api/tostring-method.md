Constructs a string giving the content of the dataset.   

**Signature** :  

- toString(): String   

**Return value** :   

- String containing the list of attribute names separated by commas   

**Parameters** :   

- none   

Example call:   

`print``("Content of dataset: " +` `dataset``.toString());`
